Compressed File Manager

History
    240325  신규 작성


개요
    압축 파일 관리를 위한 애플리케이션
    기존 압축 프로그램에서 압축파일 내부 파일 삭제 시 즉시 재압축되는 기능에 불편을 느껴 해당 문제 해결을 위해 개발
    Rust 문법 스터디 마무리 및 Rust로 DLL 생성에 대한 검증을 위해 내부 구조는 Rust로 구현
    
플랫폼
    Rust DLL(C Windows)
    C#(UI)

기능
    필수
        압축 파일을 열어 임의의 파일을 제거 한 후 재 압축 하는 프로그램
        압축 파일 내부의 파일 삭제 시 즉시 재 압축 처리 하지 않고 작업 완료 시 일괄처리 하는 것이 핵심
        압축 파일 내부의 파일 미리보기 기능(이미지)
        재압축 시 옵션(원본 유지 등)
        내부 구현 시 실제 동작을 압축 해제 후 진행할지 여부 등은 개발과정 중 처리 예정
    옵션
        압축 파일에 신규 파일 추가
